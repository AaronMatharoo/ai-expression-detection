This website needs a webcam to read facial expressions. Please visit this website on a computer instead of a mobile device!

This is expression detector! Try out different expressions in the camera to see if Expression Detector can identify the mood you are in. Try out a Happy, Surprised, or Angry expression.

The accuracy rating next to the selected expression returns the confidence level of the API on your specific expression.
